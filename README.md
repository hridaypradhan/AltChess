# AltChess
An alternate take on chess, with new objectives and a 4x4 board

This is a CodeBlocks C-language project and there are four major files:
  1. AltChessMain.c
  2. FunctionWokring.c
  3. ChessUtilities.h
  4. HelperFunctions.h
  
A project summary can be found in the file 'AltChessSummary.pdf'.
